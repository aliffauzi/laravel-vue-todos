# laravel-vue-todos
Todos with laravel and vue.js

# clone the project

```bash
git clone https://github.com/code4mk/laravel-vue-todos
```

# composer install and key generate

```bash
composer install
php artisan key:generate
php artisan migrate
```

# Node

```bash
# yarn
yarn install
yarn run dev
#npm
npm install
npm run dev
```
